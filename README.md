# 💫 About Me:
Hi im david adebanwo. Final year dev student at TUS (Limerick) who likes building things that actually get used. My main project right now is an AI Career Coach app for IT students (React + Node/Express + Postgres + OpenAI), where I’m trying to balance “smart” output with predictable, reliable behaviour. I’ve also shipped a Spring Boot + React e-commerce app. If I’m proud of anything, it’s clean structure, sensible trade-offs, and not overcomplicating stuff for the sake of it.
